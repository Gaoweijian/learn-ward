《拆掉思维里的墙》

给很久没读书的你 · 章节解读

古典 著

序言

原来我还可以这样活

你以为生活只能这样过，但其实，你永远有别的选择。

你有没有过这种感觉——明明日子过得还行，但总觉得哪里不对劲，像是被什么东西困住了，却说不清楚那是什么。古典老师说，那可能是你的"思维之墙"。

这本书从一个故事开始：一个年轻人，工作稳定，在别人眼里是"人生赢家"，但他内心却空洞得可怕。他找到古典老师，问了一个问题："我为什么不开心？"答案很简单——他一直在按照别人的剧本演戏，忘了自己才是编剧。

这本书要做的，就是帮你看清那些墙，然后一件一件地把它们拆掉。不是告诉你"要成功"，而是告诉你"你可以选择"。

这章拆的是什么墙？

  • "我只能这样"的固化思维
  • 把"社会规则"当成"唯一真理"的错觉
  • 不敢想象"另一种可能"的自我设限

拆墙工具

问自己一个神奇的问题："如果我有钱、有时间、有能力，我会做什么？"这个问题能帮你跳出"我不能"的思维惯性，看见自己真正想要的东西。先看见，才能到达。

今天就能试的一件小事

花3分钟，闭上眼睛想象一下：如果完全不考虑钱和面子，你最想过什么样的生活？不需要真的去做，只是想一想，然后把它写下来。

第一章

你是不是安全感的奴隶

房子是盔甲，也是牢笼——你省吃俭用供的不是房子，是恐惧。

这一章戳中了很多人的痛点。"安全感"这三个字，看起来很正向，但古典老师告诉你：过度追求安全感，其实是最大的陷阱。

他举了一个特别扎心的例子：有个女孩，为了"安全感"，拼命攒钱买房子。每个月还完房贷，口袋比脸还干净。她以为房子是保障，其实房子只是把她的恐惧实体化了——她不是在买房子，她是在买"不用面对未知的底气"。

更可怕的是，很多人的"稳定工作"也是同样的逻辑。铁饭碗听起来很美，但捧着铁饭碗的人，往往也捧着一颗越来越僵化的心。古典老师说了一句特别狠的话："当你全副武装地保护自己的时候，你也把整个世界挡在了外面。"

安全感不是来自外在的"铁饭碗"或"大房子"，而是来自内心的"我可以应对变化"。一个真正有安全感的人，不是拥有一切的人，而是敢于失去一部分东西、也有能力重新站起来的人。

这章拆的是什么墙？

  • "买了房才有安全感"的房子执念
  • "必须找稳定工作"的恐惧驱动型决策
  • "万一出事怎么办"的灾难化思维
  • 把安全感外包给外在条件（房子/工作/伴侣）的依赖

拆墙工具

问自己："这份'安全感'，每年要花掉我多少钱、多少可能性？"很多人从没算过这笔账。算完之后，你会发现"安全感"其实挺贵的，而且性价比未必高。

今天就能试的一件小事

拿出手机，查一下你目前的"安全感成本"——房贷、车贷、保守存款加起来是多少。然后问自己：如果没有这些压力，你会多出多少选择和可能性？不需要做决定，只是看看这个数字。

第二章

让有趣的生命扑面而来

有趣不是天赋，是对世界保持好奇的习惯。

你有没有发现，有些人好像永远对什么都提不起劲？工作是为了赚钱，恋爱是为了不孤单，旅游是为了发朋友圈……古典老师说，这种人不是"无聊"，而是"关闭"了。

有趣的反面不是无聊，而是"投入"。你对一件事不感兴趣，往往不是因为它真的无聊，而是因为你还没真正投入进去。

他举了一个特别好的例子：你看一部烂片，半小时就想走；但如果你先投入地看了前情介绍、了解人物关系，你可能会慢慢入戏，最后居然觉得还不错。兴趣这东西，有时候是需要"先投入才产生"的，而不是"先确认有趣才投入"。

古典老师还区分了"有趣"和"兴奋"——吃一顿大餐是兴奋，学一门新技能才是有趣；刷短视频是兴奋，读完一本有点难的书才是有趣。真正的有趣，往往带着一点点挑战和不确定。

这章拆的是什么墙？

  • "等我有兴趣了再去做"的被动等待
  • 把无聊归咎于事情本身，而不反思自己的参与方式
  • "这件事跟我没关系"的冷漠疏离
  • "我不会/我不行"的新事物恐惧症

拆墙工具

下次遇到"没兴趣"的事情，试着先做5分钟再说。很多人不是没兴趣，是还没进入状态。古典老师管这叫"感兴趣5分钟法则"——给自己一个承诺，先投入5分钟，之后可以选择放弃，但你往往不会。

今天就能试的一件小事

找一件你一直说"没兴趣"的事（学画画、跑步、烹饪……随便什么都行），今天只做5分钟。哪怕只是打开教程视频看了两眼，也算。用5分钟的投入，换一次"也许有点意思"的机会。

第三章

心智模式决定我们的命运

同一件事，悲观者看到的是墙，乐观者看到的是门。

这是全书最烧脑、也最重要的一章。古典老师引入了"心智模式"这个概念——简单来说，就是你看待世界的那副"眼镜"。

同一个半杯水，悲观者看了会想"只剩半杯了"，乐观者会想"还有半杯呢"。杯子没变，但你看到的东西不一样了。心智模式，就是决定你看什么的那副眼镜。

更厉害的是，心智模式不仅是"看"，还会"创造"。你以为"我很笨"，你就会避免尝试；你避免尝试，就真的没进步；没进步，你就更确信"我很笨"——看，一个完美的自我封闭循环形成了。这不是命运，这是你自己挖的坑。

古典老师介绍了塞利格曼的"习得性无助"实验：把狗关在笼子里电击，时间长了就算打开笼门，狗也不跑了——它学会了"无助"。人也一样。如果你从小被说"笨"，被说"不是读书的料"，你可能早就学会了"无助"，只是你自己不知道。

好消息是：既然心智模式是"学会"的，它也可以被"重新学"。拆掉旧眼镜，换一副新的，世界就变了。

这章拆的是什么墙？

  • "我就是这样的人"的自我固化
  • 把过去经历形成的"结论"当成永久真理
  • "世界就是这样的"的狭隘认知
  • 负面事件后的"灾难化解读"习惯

拆墙工具

质疑你的"绝对化信念"。当你冒出"我永远都……""我肯定做不到……""所有人都会觉得……"这类想法时，在心里给它加个问号。试试改成"目前还没""也许""有些人可能会"——语言一变，局限就松了。

今天就能试的一件小事

留意一下今天你说出口或想到的"绝对化句式"（比如"我总是……""我从来不……""这事肯定不行……"）。把它写下来，然后尝试改写成一个不带"永远"和"肯定"的版本，看看这个变化带来的感受。

第四章

成功学不能学

复制别人的成功路径，得到的是别人的结果，不是你的。

这一章简直是"成功学鸡汤"的大型翻车现场。古典老师毫不客气地说：成功学教你的，全是废话。

为什么？因为成功学最核心的逻辑就是"复制成功者"。但问题是，每个成功者的路径都依赖特定的时间、地点、个人特质和运气。你复制了他的动作，但复制不了他当时的天时地利人和。结果就是：你学得越像，离成功越远，因为你在用昨天的地图走今天的路。

他举了一个例子：马云说"我不看学历"，因为他是马云——他已经是马云了。对一个刚毕业的人来说，不看学历这条建议，等于废话。成功者的经验往往是他成功之后回头看的总结，不是他成功之前的行动指南。

古典老师还戳穿了一个更大的谎言："只要努力就能成功"。他说，努力是成功的必要条件，但绝对不是充分条件。方向错了，跑得越快离目标越远。

那应该怎么办？答案是：与其复制别人的成功路径，不如搞清楚"什么对你来说是成功"。你想成为谁，你想过什么生活——这才是你的答案，别人的答案永远是你的毒药。

这章拆的是什么墙？

  • "成功一定有方法"的简单化思维
  • "模仿成功者就能成功"的复制执念
  • "只要努力就会成功"的线性因果幻觉
  • "别人那样做成功了，我也可以"的盲目跟从

拆墙工具

停止模仿，开始研究。把你崇拜的人当成"案例"而不是"模板"——研究他为什么这样做，而不是照着他做。研究背后的逻辑，而不是表面的动作。

今天就能试的一件小事

找一个你一直想模仿的"成功人士"，今天花10分钟查一下他的背景：他是哪年入行的？他遇到最大的挫折是什么？你对他的了解，和你最初以为的有什么不同？

第五章

如何找到热爱的职业

不是"找到"热爱，而是"活出"热爱。

这一章专门回答了一个让无数人头疼的问题："我到底适合做什么？"

古典老师首先拆掉了一个幻觉："等我找到梦想再出发"。很多人把"找到梦想"当成开始行动的借口——"等我找到真正热爱的事情，我就努力。"但问题是，你坐在家里想，是想不出梦想的。梦想是在行动中逐渐清晰的。

他提出了一个很实用的概念：职业发展有三个阶段——生存期（为了钱）、发展期（为了能力）、事业期（为了意义）。很多人搞反了，还没解决生存问题就追求意义，结果两头都不稳。

更颠覆认知的是：职业选择不是"我适合做什么"，而是"我愿意为什么付出代价"。古典老师说，你愿不愿意为一件事"吃亏"，才是检验你是否真正热爱的标准。你说喜欢写作，但如果让你免费写三年你还愿意吗？你说喜欢画画，但如果卖不出去你还画吗？热爱不是顺境时的喜欢，是逆境中的坚持。

这章拆的是什么墙？

  • "等我找到梦想再行动"的拖延借口
  • "一定有一份完美工作等着我"的幻想
  • "我要 passion！"的激情崇拜（其实激情是做了之后才来的）
  • "我不知道我要什么"的无助感（你只是还没开始探索）

拆墙工具

用"代价测试"代替"喜欢测试"。不要问自己"我喜欢什么"，问自己"我愿意为它付出什么代价"。你愿意早起吗？愿意免费做吗？愿意被拒绝吗？愿意坚持3年没有回报吗？愿意，才是真的喜欢。

今天就能试的一件小事

写下三个"我想做的事"，然后问自己：如果做这件事需要免费做两年，你还愿意吗？如果答案是"不愿意"，那可能只是喜欢，不是热爱；如果答案是"愿意但有点犹豫"，这才是值得认真对待的信号。

第六章

原来我还可以这样活

你的人生脚本，不是别人写的，是你自己默许的。

这章是全书最"催泪"的一章。古典老师讲了一个故事，主人公是典型的"别人家的孩子"——名校毕业、进外企、年薪几十万人人羡慕。但他找到古典老师时，说了一句话："我从来没有为自己做过一个决定。"

这句话太重了。你想想，从小学什么兴趣班，到高考选什么专业，到毕业找什么工作，到什么时候结婚……这些决定，有多少是你自己做的？有多少是父母替你做的、社会替你做的、别人替你做的？

古典老师没有责怪父母的意思。他指出的是：很多人已经成年了，但内心还住着一个害怕父母不高兴的小孩。他们做的每一个选择，都是"父母会怎么看"，而不是"我想要什么"。

更残酷的是：你的父母可能并不像你以为的那样强硬。当你鼓起勇气说出"这是我自己的选择"时，你可能会惊讶地发现——父母比你想象的更能接受你的真实。这个世界没有那么不允许你做自己，是你自己把自己关了起来。

这章拆的是什么墙？

  • "如果我做了不一样的选择，爸妈会伤心"的愧疚捆绑
  • "大家都这样，所以我也要这样"的社会惯性
  • "我不敢"的长期自我压制
  • 把"被认可"看得比"做自己"更重要的价值错位

拆墙工具

练习"区分责任"。问自己：这件事的后果，是谁在承担？如果是你自己承担，那就大胆做；如果是父母承担，你才需要认真沟通。不要用"父母会伤心"来逃避自己应该承担的责任，也不要用"我做自己"来逃避沟通。

今天就能试的一件小事

找一个最近让你纠结的决定，问自己：如果完全不考虑任何人的意见，你会怎么选？把那个答案写在纸上。不需要真的按它做，只是看看那个"真正的自己"说了什么。

第七章

幸福是一种转换力

幸福不是得到更多，而是看到你已经拥有的东西。

这一章谈的是幸福。听起来很俗，但古典老师讲出了新意。

他提出了一个概念叫"受害者和掌控者"。面对同一件坏事，受害者的反应是"为什么偏偏是我"；掌控者的反应是"我能从中学到什么"。这不是阿Q精神，这是完全不同的心智模式。

古典老师分享了一个自己的故事：他曾经被一家公司放鸽子，损失了很多钱。换成以前的他，可能会崩溃、抱怨。但那次，他选择把这个经历写成文章，后来这篇文章帮助了很多人。这就是"转换力"——把挫折转换成资源，把问题转换成答案。

他还提到了"父母幸福观"的问题。很多人的不幸福感，来自"父母的期待"和"自己的渴望"之间的撕裂。父母觉得稳定的体制内工作才是幸福，你自己却渴望不一样的生活。古典老师说：父母有父母的幸福，你有你的。你不需要用父母的幸福标准，来衡量自己的快乐。

这章拆的是什么墙？

  • "为什么我这么努力还是不开心"的单向思维
  • "我要XXX才会幸福"的幸福延迟症
  • 把幸福外包给外在条件（财富/地位/他人认可）
  • "凭什么是他不是我"的受害者心态

拆墙工具

每天写"三件好事"。不是大事，就是三件小到不值得记录的事：今天阳光很好、喝到一杯好喝的咖啡、地铁上有座位。科学研究证明，持续一个月，你大脑的幸福感知力会显著提升。这不是心灵鸡汤，这是神经可塑性。

今天就能试的一件小事

今天睡前，拿出手机备忘录，写下今天发生的三件好事。哪怕是"吃到了好吃的午饭"这种小事也行。不用文采飞扬，只要写下来。

第八章

上天不需要你成功，只需要你尝试

完成比完美重要，开始比完成重要。

这一章是关于"开始"的。很多人卡在"想"和"做"之间，一辈子出不来。古典老师把这叫"开始恐惧症"。

他的核心观点是：你不需要准备好了才开始，你需要开始了才准备好。很多人以为要"先把一切都规划好"再行动，但真实世界不是这样的。世界是由那些"先跳下去"的人改变的，不是由那些"在岸上计算最优路线"的人。

他还提到了一个悖论：你越追求完美，越难开始；你越开始，越发现完美根本不存在。很多人写了一辈子小说第一章，就是因为他们觉得"第一章必须完美"。结果，一辈子过去了，最完美的一章从来没有存在过。

古典老师给出的建议是"先失败，再成功"。先做，先错，先丢人，然后你才知道哪里要改。成功不是一蹴而就的，成功是一次又一次失败的迭代。真正的勇者，不是从不失败的人，而是摔倒了还能爬起来继续跑的人。

这章拆的是什么墙？

  • "等我准备好了再开始"的完美主义拖延
  • "万一失败了怎么办"的失败恐惧
  • "第一次就要做好"的不合理标准
  • "做不好不如不做"的自我保护过度

拆墙工具

给自己设一个"丢人额度"。每个月允许自己失败3次、丢脸3次。你会发现，真正的大灾难几乎不会来，最坏的结果不过是一点点尴尬，而尴尬是成长最便宜的学费。

今天就能试的一件小事

找一件你一直想做但没做的事，今天就迈出第一步。不需要完成，不需要做好，只需要开始：下载那个App、打开那个教程、注册那个账号、约那个人喝咖啡。开始，就赢了。

后记

走在期待的路上

人生没有标准答案，但有值得认真对待的活法。

后记里，古典老师回顾了自己的心路历程。他曾经也是那个"按照社会剧本演戏"的人，做着人人羡慕的工作，内心却空得要命。

他写道：拆墙不是拆掉所有的墙——有些墙是为了保护你而存在的。拆墙的意义在于：让你知道哪些墙是真的需要，哪些墙只是习惯和恐惧。

读到这里，你可能会发现：这本书最大的价值，不是告诉你"应该怎样活"，而是让你看见"原来我还可以这样活"。看见，就是改变的第一步。

— 致 很久没读书的你 —

给很久没读书的你的一封信

你好。

谢谢你读到这里。我知道，注意力是这个时代最稀缺的资源，你能从头到尾读完一本书，本身就已经是一件了不起的事——哪怕你中间跳过了几段，哪怕你偶尔走神又回来了。你在这里，这就是最重要的。

这本书叫《拆掉思维里的墙》，但说到底，它想说的只有一件事：你比自己以为的更自由。

那些"应该"和"只能"，那些"别人都这样"和"我一直如此"——它们不是枷锁，只是墙。而墙，是可以被拆掉的。

1. "你永远有别的选择。"

2. "完成比完美重要，开始比完成重要。"

3. "幸福不是得到更多，而是看到你已经拥有的。"

这三句话，不需要背，不需要理解透，只需要记住。等你哪天被困住的时候，它们会冒出来的。

最后，我想邀请你做一件事：从今天开始，每天读5分钟书。不多，就5分钟。一个月后，你会惊讶于自己竟然读完了好几本书。

阅读不是任务，阅读是给脑子放假。当你沉浸在别人的故事和思考里，你会发现自己的那些烦恼，好像也没那么大了。

慢慢来，没关系。重要的是，你已经在路上了。

——你的读书向导
